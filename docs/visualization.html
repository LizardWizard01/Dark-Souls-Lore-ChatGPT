This network functions as a way to uncover information on what Natural Language Processing(NLP) programs like spaCy can accomplish. Our network focusing on gwyn allows one to see the association bewteen him and these items within Dark Souls. Using this analysis, we can interpret how AI chatbots read and understand this data, allowing for further speculation. Our project started as an idea to have chatGPT use in-game item descriptions to generate original, lore-friendly items. Seeing how AI interprets the data confirms that this is possible as it's able to make connections bewteen descriptions. Using AI can also be useful for people trying to understand some of the more hidden lore bits from Dark Souls, as the AI points out connections. Seeing Gwyn's relation to each item gives us a unique prespective.
